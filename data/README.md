# data/

This folder is reserved for partner-specific or restricted data that cannot be stored in the shared `Data/` folder.

Shared data sources (SEC filings, EIA, DOE LEAD, ACS, GIS) are referenced via relative paths to `../../Data/` and `../../Cleaned_Data/`. Do not duplicate shared data here.

If partner-specific data is added to this folder, document it here with: source, date received, and why it cannot go in the shared `Data/` folder.
